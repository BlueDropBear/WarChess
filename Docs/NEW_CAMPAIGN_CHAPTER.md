# WarChess — Act IV: The Hundred Days

## Overview

Act IV is the first post-launch campaign expansion, covering Napoleon's return from exile and the final battles leading to Waterloo. It assumes the player has mastered all 14 base units and introduces 3 new units (Jäger, Supply Wagon, Uhlan) alongside advanced terrain combinations.

**Act IV is included in the Full Campaign purchase** (no additional cost).

---

## Narrative Arc

After Napoleon's exile to Elba, Europe believed the wars were over. But the Emperor escaped, rallied his veterans, and marched on Paris. The Hundred Days had begun. You command the coalition forces racing to stop Napoleon before he can reconquer Europe.

## Star Rating Thresholds

Same as existing campaign:
- 1 Star: Victory
- 2 Stars: Decisive victory (50%+ units survive)
- 3 Stars: Flawless (all units survive)

Battle 38 (Retreat) special: stars based on units surviving at round 15.

## Implementation Notes

### CampaignDatabase.cs
- Add battles 31-40 following existing pattern
- Battle 38 needs new win condition: `SurvivalRounds = 15`
- Battle 40 needs reinforcement mechanic: `ReinforcementsAtRound = 10`

### New CampaignBattleData Fields
```csharp
public int SurvivalRounds;           // 0 = normal win condition, >0 = survive N rounds
public int ReinforcementsAtRound;     // 0 = none, >0 = spawn allies at this round
public List<EnemyUnitPlacement> Reinforcements; // Units to spawn as reinforcements
```

### BattleEngine Changes
- Handle survival win condition (check round count instead of enemy elimination)
- Handle mid-battle reinforcements (spawn units at specified round)
